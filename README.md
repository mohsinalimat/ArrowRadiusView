## Installation

ArrowRadiusView is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:
```ruby
use_frameworks!
pod 'ArrowRadiusView' :git => 'https://github.com/zengxianshu/ArrowRadiusView.git'
```
